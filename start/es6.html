      # 2.1. ES5

      当前 JavaScript 标准，兼容几乎所有浏览器。

      # 2.2. ES6

      下一代 JavaScript 标准，2015 年 6 月正式发布。

      TIP

      2015 年开始，JavaScript 标准在每年 6 月正式发布一次，因此 ES6 包含了 ES2015、ES2016、ES2017。

      # 2.3. TypeScript

      微软 2012 年推出的一种编程语言，属于 JavaScript 的超集，需要编译为 JavaScript 执行。它的最大特点就是支持强类型和 Class。
